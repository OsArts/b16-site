# Project b16-site

## Project setup

- Check `node -v` (or install `Node` |> use `nvm` )
- `npm install`
- `./scripts/download_videos.sh` << Run script for download videos [seed assets]

- `npm run serve`

### Compiles and minifies for production

- `npm run build`

### Lints and fixes files

- `npm run lint`
